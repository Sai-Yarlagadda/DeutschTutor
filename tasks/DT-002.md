## Description

Tailwind is installed by the scaffold (DT-001) but needs a custom design system configured for DeutschTutor — dark/moody theme fitting an immersion experience (not bright and gamified).

---

## Acceptance Criteria

- [ ] `tailwind.config.ts` extended with custom color palette:
  - Background: deep charcoal / near-black
  - Accent: warm amber (German flag warmth)
  - Text: off-white
- [ ] Custom font configured (suggestion: `Inter` for UI, `Lora` for German text display)
- [ ] `globals.css` has base layer resets
- [ ] A sample page renders correctly using only Tailwind utility classes
- [ ] No unused CSS warnings

---

## Design Tokens

```ts
// tailwind.config.ts — extend colors

  background: '#0f0f0f',

  }
```
