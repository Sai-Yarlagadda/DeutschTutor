## Description

Wire up the Claude API to power the AI persona. The Persona Engine takes a user transcript + persona config + conversation history and returns a natural German response in character.

---

## Acceptance Criteria

- [ ] `@anthropic-ai/sdk` installed
- [ ] API route `POST /api/chat` created at `src/app/api/chat/route.ts`
- [ ] Route accepts:

```ts
{
  persona: {
    name: string;
    age: number;
    city: string;
    occupation: string;
    scenario: string;
  };
  userMessage: string;
  history: { role: 'user' | 'assistant'; content: string }[];
  userLevel: string; // e.g. "A2"
}
```

- [ ] Route returns:

```ts
{ reply: string }
```

- [ ] System prompt uses the template from `implementation.md`:

```
You are {persona.name}.
You are from {persona.city}.
You work as {persona.occupation}.
You are currently in {persona.scenario}.

Speak naturally. Remain in character. Use realistic German.
Never switch to English. Adapt to learner level: {userLevel}.
Correct mistakes naturally inside conversation.
```

- [ ] Response is streamed (use `StreamingTextResponse` or `ReadableStream`) so it appears word-by-word in the UI
- [ ] Model: `claude-sonnet-4-6` (latest capable model, not Haiku)
- [ ] Errors returned as `{ error: string }` with appropriate HTTP status
