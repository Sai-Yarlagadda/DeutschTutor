## Description

Build the `/personas` page where the user picks who they will talk to. Only personas compatible with the selected scenario are shown (filtered from DT-015 data).

---

## Acceptance Criteria

- [ ] Page at `/personas?scenario={id}`
- [ ] Reads `scenarioId` from URL query param
- [ ] Filters the persona library to only show personas whose `scenarioId` matches
- [ ] Each persona card shows:
  - Name (large)
  - Age + city (muted subtitle)
  - Occupation
  - Region badge (e.g. "Bayern 🇩🇪")
  - Voice type label (e.g. "Professional · Female")
- [ ] Clicking a card saves `personaId` and navigates to `/conversation?scenario={id}&persona={id}`
- [ ] "Back" button returns to scenario selection
- [ ] If no personas match the scenario: show a friendly "Keine Personas verfügbar" message
