## Description

Add looping background audio that plays during conversations to enhance immersion. Each scenario has its own ambient soundscape.

---

## Acceptance Criteria

### Audio Files

- [ ] Source or record royalty-free ambient audio for each scenario:

| Scenario | File | Description |
|----------|------|-------------|
| Railway Station | `public/audio/railway.mp3` | Train sounds, announcements, crowd |
| Café | `public/audio/cafe.mp3` | Coffee machine, chatter, soft music |
| Hotel | `public/audio/hotel.mp3` | Lobby ambience, soft piano |
| University | `public/audio/classroom.mp3` | Lecture hall murmur, distant chatter |
| German Friend | `public/audio/apartment.mp3` | Quiet room, street sounds |

- [ ] All files are MP3, under 2MB each, loop seamlessly

### `useAmbientAudio` Hook

- [ ] Created at `src/hooks/useAmbientAudio.ts`
- [ ] API:

```ts
const { play, pause, setVolume, isPlaying } = useAmbientAudio(audioFile: string);
```

- [ ] Loops continuously using Web Audio API or HTML `<audio loop>`
- [ ] Default volume: 0.25
- [ ] Fades in over 2 seconds on start
- [ ] Fades out over 1 second on stop
- [ ] Pauses automatically when AI TTS is speaking (optional, duck/lower volume instead)

### Fallback

- [ ] If audio file fails to load, fail silently (no error shown to user)

---

## Audio Sources (royalty-free)

- Freesound.org
- Pixabay audio
- ZapSplat

---

## Definition of Done

Open conversation screen → ambient audio begins playing at low volume. Matches the chosen scenario. Loops without audible gap.
