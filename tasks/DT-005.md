## Description

Capture the user's microphone input and convert spoken German to text. The transcript feeds into the Persona Engine (DT-006).

Two viable approaches — choose one:

| Option | Pros | Cons |
|--------|------|------|
| **Web Speech API** (browser-native) | Free, zero latency, no API key | English-only on some browsers; no guaranteed German support in all envs |
| **OpenAI Whisper API** | Excellent German, reliable | ~$0.006/min cost; requires API key |

**Recommendation:** Use Web Speech API with `lang="de-DE"` first. Fall back to Whisper if quality is poor.

## Acceptance Criteria

- [ ] `useSpeechRecognition` custom hook created at `src/hooks/useSpeechRecognition.ts`
- [ ] Hook exposes: `{ transcript, isListening, startListening, stopListening, error }`
- [ ] Language set to `de-DE`
- [ ] Works in Chrome (primary target)
- [ ] Interim results shown in UI while user speaks (live transcript preview)
- [ ] API route `POST /api/transcribe` created as fallback for Whisper (even if not wired up yet)
- [ ] Output matches the spec:

```json
{ "transcript": "Wo fährt der Zug nach Berlin ab?" }
```

## Implementation Notes

```ts
// src/hooks/useSpeechRecognition.ts
const recognition = new (window.SpeechRecognition || window.webkitSpeechRecognition)();
recognition.lang = 'de-DE';
recognition.interimResults = true;
recognition.continuous = false;
```

For the Whisper fallback route:
```ts
// src/app/api/transcribe/route.ts
// POST: receives FormData with audio blob, calls OpenAI Whisper, returns { transcript }
```
