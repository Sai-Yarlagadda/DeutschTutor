## Description

Define the 4 core MVP personas as typed data. These are the characters the user can talk to. Each persona maps to a scenario, a Claude character prompt, and an ElevenLabs voice.

---

## Acceptance Criteria

- [ ] Persona data defined in `src/lib/personas.ts`
- [ ] Schema:

```ts
export type Persona = {
  id: string;
  name: string;
  age: number;
  city: string;
  region: string;
  occupation: string;
  scenarioId: string;
  voiceType: 'professional-female' | 'professional-male' | 'young-male' | 'older-female';
  voiceId: string;           // ElevenLabs voice ID
  personality: string;       // 2-3 sentences injected into system prompt
  languageLevel: 'native' | 'standard' | 'dialect';
  dialect?: string;          // e.g. "Bavarian", "Austrian"
};
```

- [ ] Four personas defined:

| ID | Name | Age | City | Occupation | Scenario | Voice |
|----|------|-----|------|------------|----------|-------|
| `railway-employee` | Anna | 32 | Hamburg | Railway Employee | `railway-station` | Professional female |
| `berlin-student` | Lukas | 24 | Berlin | University Student | `university` | Young male |
| `bavarian-grandma` | Helga | 68 | Munich | Retired | `german-friend` | Older female |
| `austrian-receptionist` | Sophie | 28 | Vienna | Hotel Receptionist | `hotel` | Professional female |

- [ ] `getPersona(id: string): Persona` helper exported
- [ ] `getPersonasByScenario(scenarioId: string): Persona[]` helper exported
- [ ] Personality strings are vivid and character-defining (not generic)

---

## Example Personality Strings

```ts
// Anna
"Anna is efficient and warm but always professional. She knows every platform at Hamburg Hauptbahnhof by heart. She takes pride in keeping things on schedule."

// Lukas
"Lukas is laid-back, uses a lot of current Berlin slang, and is always late to lectures. He talks fast and makes pop culture references."

// Helga
"Helga speaks with a thick Bavarian accent, loves talking about her garden and her grandchildren. She is patient, warm, and speaks slowly and clearly."

// Sophie
"Sophie is polished and professional with a soft Viennese accent. She is efficient but genuinely friendly. She'll sprinkle in Austrian vocabulary like 'Bitte sehr' and 'Grüß Gott'."
```

## Definition of Done

`getPersonasByScenario('railway-station')` returns Anna. All 4 personas load with no TypeScript errors.
